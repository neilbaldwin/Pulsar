Commands U - Z

Uxx Note Probability ABCDE Pattern

When placed on the same step as a note, makes the playback of that note dependent on a weighted random number. The higher the value of XX the less likely the note will play.

Vxx --- Reserved For Future Feature --- ----- Unused


Wxx Set Duty or Duty Table AB--- Pattern/Table

Overrides the DUTY setting of the current Instrument.

First four values are small icons that represent a fixed pulse width (duty). Subsequent values are Duty Table numbers starting at 00

Xxx --- Reserved For Future Feature --- ----- Unused


Yxx Echo AB-D- Pattern/Table

Overrides the ECHO setting of the current Instrument

XX is an Echo program from the Echo List.

Zxx Table Speed ABCDE Pattern/Table

Sets the speed of any Tables that are run on the Track.

The default Table Speed is set to 80 which is the middle of the speed range, 02 being slowest and FF being the fastest.

00 has a special function in that it synchronises the Table with the Pattern. This means that the Table will advance to the next step each time the Pattern advances a step. Table is restarted on each new note.

01 has a special function in that it synchronises the Table with the Pattern. This is similar to '00' but the table is not restarted on each new note.
