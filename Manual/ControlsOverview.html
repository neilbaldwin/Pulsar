Overview of Controls

This page is intended as a quick reference for all the controls in Pulsar. To get further information and a better explanation of each control, click the links in the left side of the table.


Basic Controls
To move around editor pages tap/hold U/D/L/R
To jump to a specific editor pagetap B on a parameter
To jump to any editor pagehold B then tap DOWN
To jump back to previous editor pagehold B then tap LEFT
To go to Setup/Utility pagehold B then tap UP
To modify a valuehold A then tap/hold U/D/L/R
To "erase" a valuehold SELECT then tap B
To quickly enter last value againtap A
To play the current Song from the starthold SELECT then tap START
To play the current Song from the cursortap START
To stop the current Songtap START while Song is playing
Advanced Controls 1 - More Editing
To insert next unused Chain or Patterndouble-tap A
To clone a Chain or Patternhold SELECT then tap A
To toggle a note between 'normal' and 'tie'hold SELECT then tap A (in Pattern)
To delete an entire rowhold B+A then tap UP
To insert a rowhold B+A then tap DOWN
Advanced Controls 2 - Object Navigation
To change the focused Track from within any editor pagehold SELECT+B then tap L/R
To traverse Song/Chain from within Chain/Pattern editorhold SELECT+B then tap U/D
To scroll through all Objects in editor pagehold SELECT then tap U/D
Advanced Controls 3 - Mute & Solo
To mute/unmute the current Track from within any editor pagehold B then tap SELECT
To solo/unsolo the current Track from within any editor pagehold B then tap START
Advanced Controls 4 - Copy & Paste
To copy whole or part of an objecthold SELECT then tap LEFT
To paste data from the Copy Buffer into an objecthold SELECT then tap RIGHT
Advanced Controls 5 - Transpose
To Block Transpose all values in a columnhold A+SELECT then tap L/R
To Smart Transpose all values in a columnhold A+SELECT then tap U/D
